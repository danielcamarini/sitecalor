O que é dimerização?


A dimerização é o processo de controle da intensidade de brilho de um dispositivo, como um LED. Em vez de apenas ligar ou desligar, a dimerização permite ajustar o nível de luminosidade de forma gradual, deixando a luz mais fraca ou mais forte conforme necessário. Esse controle é geralmente realizado por meio de uma técnica chamada PWM (modulação por largura de pulso), que varia o tempo em que o sinal permanece ligado dentro de um intervalo.


Apresente um exemplo de dimerização com o LED a partir de um programa temporizado (a cada 100 ms)


int led = 9;
int brilho = 0;
int incremento = 5;
void setup() {
pinMode(led, OUTPUT);
}
void loop() {
analogWrite(led, brilho);
brilho = brilho + incremento;
if (brilho <= 0 || brilho >= 255) {
incremento = -incremento;
}
delay(100);
}


Apresente um exemplo de dimerização com o LED a partir de comandos enviados por botão do tipo “push button”


int led = 9;
int botao = 7;
int brilho = 0;
int incremento = 5;
bool estadoAnterior = HIGH;
void setup() {
pinMode(led, OUTPUT);
pinMode(botao, INPUT_PULLUP);
}
void loop() {
bool estadoBotao = digitalRead(botao);
// Detecta o clique
if (estadoBotao == LOW && estadoAnterior == HIGH) {
brilho += incremento;
if (brilho > 255) {
brilho = 255;
}
analogWrite(led, brilho); // aumenta o brilho do LED
delay(200); // debounce
}
estadoAnterior = estadoBotao;
}
